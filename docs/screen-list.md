# DogWalk 화면 목록 (MVP)

## 1) 사용자 앱(견주)
1. 스플래시
2. 로그인/회원가입
3. 약관/권한동의
4. 홈 대시보드
5. 반려견 목록
6. 반려견 등록/수정
7. 대리산책 예약 생성
8. 결제 수단 선택/결제
9. 예약 완료/매칭 상태
10. 실시간 산책 지도
11. 산책 완료 리포트
12. 셀프 산책 시작/진행/종료
13. 산책 기록 목록/상세
14. 크레딧 지갑
15. 쇼핑몰 목록/상세/장바구니/주문
16. 결제 내역
17. 내 정보/고객센터

1. 로그인/회원가입
2. 자격증 업로드/심사상태
3. 홈(온라인 상태 토글)
4. 요청 목록
5. 요청 상세/수락
6. 이동 시작/도착
7. 산책 시작/진행/종료
8. 산책 사진/메모 업로드
9. 정산 내역
10. 리뷰/평점 확인
11. 프로필 관리

1. 로그인

4. 워커 심사(승인/반려)

6. 결제 조회/실패건
7. 환불 처리
8. 정산 배치/정산 내역
9. 크레딧 정책/수동조정
